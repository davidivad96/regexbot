# RegexBot

Using AI to help you generate regular expressions using natural language.

## Description

RegexBot is a web application that uses AI to help you generate regular expressions using natural language.
The app uses the Vercel AI SDK to generate regular expressions based on natural language input.

## Features

- Generate regular expressions from natural language

## Tech Stack

- Next.js 14
- Vercel AI SDK
- DaisyUI
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Set the environment variables (use the `.env.example` file as a template)
3. Install the dependencies with `pnpm install`
4. Run the development server with `pnpm dev`
5. Open http://localhost:3000 with your browser and start using the app!
